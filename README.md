# 📊 Olist E-commerce | End-to-End Data Analysis Project

## 🚀 Overview
Este projeto consiste em uma análise completa de dados de e-commerce utilizando o dataset público da Olist (Kaggle), passando por todo o pipeline de dados desde a ingestão até a visualização estratégica.

O objetivo foi transformar dados brutos em insights analíticos, simulando cenários reais de negócio e apoiando a tomada de decisão orientada por dados.

## 🧠 Objetivo do Projeto
- Aplicar conceitos de **Data Analytics e Business Intelligence**
- Construir um pipeline completo de dados (**ETL**)
- Desenvolver **KPIs estratégicos**
- Criar dashboards interativos com foco em **Data Visualization e Storytelling**

---

## ⚙️ Pipeline de Dados

### 🔹 Fonte de Dados
- Dataset: Olist E-commerce (Kaggle)
- Formato: CSV

### 🔹 ETL (Extract, Transform, Load)
- Ferramenta: Pentaho (PDI)
- Processos:
  - Ingestão de dados
  - Limpeza e transformação (Data Cleaning & Data Transformation)
  - Padronização de dados
  - Carregamento no banco

### 🔹 Banco de Dados
- PostgreSQL
- Modelagem relacional
- Estruturação e organização das tabelas
- Ajustes via SQL

---

## 📊 Business Intelligence (Power BI)

### 🔹 Modelagem
- Criação de relacionamentos
- Tabela calendário

### 🔹 Transformação
- Power Query (M Language)

### 🔹 Métricas
- DAX (Data Analysis Expressions)

### 🔹 Visualização
- Dashboards interativos com drill-down
- Navegação por diferentes níveis de granularidade

---

## 📈 Principais Análises

- **Faturamento | Pedidos | Ticket Médio | Clientes Únicos**
- **Evolução do Faturamento ao longo do tempo**
- **Análise por Status e Tipo de Pagamento**
- **Produtos e Categorias (Ranking + Curva de Pareto 80/20)**
- **Indicadores Logísticos (Frete e Tempo de Entrega)**
- **Análise Geográfica e Distribuição de Vendas**

---

## 🎯 Destaques

- Pipeline completo de **ETL**
- **Análise Exploratória de Dados (EDA)**
- Criação de **KPIs estratégicos**
- Aplicação de **Pareto (80/20)**
- **Storytelling com dados**
- Design visual com Illustrator e Photoshop

---

## ⚠️ Observações

- Produtos representados por **IDs**, devido à ausência de nomes na base pública
- Dados tratados e padronizados para garantir consistência nas análises
